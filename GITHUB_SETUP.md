## Các bước upload

1. Vào GitHub và chọn **New repository**.
2. Nhập repository name và description ở trên.
3. Chọn **Public**.
4. Không chọn tạo sẵn README, `.gitignore` hoặc license vì bộ file này đã có README và `.gitignore`.
5. Chọn **Create repository**.
6. Giải nén file `vietnam-omnichannel-retail-analytics.zip` trên máy.
7. Trong repository, chọn **Add file → Upload files**.
8. Kéo toàn bộ file và thư mục bên trong thư mục đã giải nén vào cửa sổ upload.
9. Nhập commit message:

```text
Initial project upload
```

10. Chọn **Commit changes**.

File Power BI hiện khoảng 19 MB nên vẫn nằm dưới giới hạn 100 MB cho mỗi file của GitHub.

## Sau khi upload

- Mở phần **About** bên phải repository và thêm description.
- Thêm các topics ở trên.
- Ghim repository này lên đầu trang GitHub profile bằng **Customize your pins**.
- Kiểm tra README có hiển thị ảnh dashboard và các đường dẫn tới file PBIX/PDF hoạt động hay không.
