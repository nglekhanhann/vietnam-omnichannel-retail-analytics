# SQL analysis

The SQL workflow was completed with DuckDB and covered:

- data profiling and duplicate checks
- cleaned views and business-rule corrections
- KPI overview
- monthly, MoM, and YoY performance
- channel and store performance
- category and SKU profitability
- promotion effectiveness
- return and stockout analysis
- product ranking and ABC classification

The analytical decisions, outputs, and interpretations are documented in `docs/retail_analytics_project_report.pdf`.

The original notebook or `.sql` scripts were not part of the uploaded project files, so they are not included in this repository package. Add them to this folder later to make the analysis fully reproducible.
