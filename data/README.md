# Data

The raw CSV files are not included in this repository package.

The project uses a synthetic Vietnamese omnichannel FMCG retail dataset covering 2024–2025. The original model contains:

- `retail_sales_transactions` — transaction line-item fact table
- `retail_product_master` — product dimension
- `retail_store_master` — store dimension
- `retail_customer_master` — customer dimension
- `retail_calendar` — date dimension
- `retail_promotion_calendar` — promotion dimension

The prepared data model is embedded in the Power BI file under `powerbi/`.
